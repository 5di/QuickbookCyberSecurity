## Anonymous-OS {#anonymous-os}

Anonymous已经是大名鼎鼎了吧，最近Anonymous发布了命名为Anonymous-OS的操作系统，基于Ubuntu 11.10，提供各种web渗透测试工具，包括：

- ParolaPass Password Generator

- Find Host IP

- Anonymous HOIC

- Ddosim

- Pyloris

- Slowloris

- TorsHammer

- Sqlmap

- Havij

- Sql Poison

- Admin Finder

- John the Ripper

- Hash Identifier

- Tor

- XChat IRC

- Pidgin

- Vidalia

- Polipo

- JonDo

- i2p

- Wireshark

- Zenmap

列出的只是一部分而已，还有更多工具在其中。
